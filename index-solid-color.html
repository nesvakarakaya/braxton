QPFace Mobile Application

QPFace is a mobile application that captures biometric facial photos of users and analyzes their facial expressions to provide insights into their characteristic traits. I have developed and integrated a custom prompt using ChatGPT 3.5 into a Python script, which interprets the facial expressions to offer personalized feedback. User data is securely stored on Azure servers, utilizing Azure NoSQL (Cosmos DB) for the database. The backend development is done using Python, and I am also involved in the UI/UX design and development processes.

Andy Warhol Project

Inspired by Andy Warhol's art, this C# Windows Forms application analyzes portrait photos of gallery visitors to provide an artistic interpretation. It uses Python and OpenCVSharp to analyze facial features, determine personality tendencies, and store the results in a JSON file. The application includes facial recognition, photo capture, and personality analysis functionalities. It selects a suitable artwork from an SQL Server database based on the analysis results and saves the details in SQL.
